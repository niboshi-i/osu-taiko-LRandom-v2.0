osu!taiko LRandom v2.0 read me



～概要～

このツールでは、osu!taikoのノーツを本家のオプション"でたらめ"や、でたらめの親戚"制限付きでたらめ"に変換することができます

制限付きでたらめとは・・・同じ色のノーツの連続が指定した長さからランダムで選ばれるようになります、他にいい名前があれば教えてください><



～使い方～

１、"Songsフォルダーのパス"の横にある参照ボタンを押してSongsフォルダーを指定してください
.osuファイルを参照するときにこのフォルダーを開きます
設定しなくも問題ないです

２、"生成したい譜面の.osuファイル"の横にある参照ボタンを押して.osuファイルを指定
もしくは曲選択画面で、譜面を右クリックし、「編集」を選択して、Edit画面のメニューバーから「ファイル」>「フォルダーを開く」を選択して、.osuファイルをこのソフトにドラッグ＆ドロップしてください

３、好みの設定に変更して"実行"を押してください、選んだ譜面があるフォルダに自動で追加されます



～設定～

"でたらめ"　面(ドン)と縁(カッ)がランダムで登場するようになります

"制限付きでたらめ"　同じ色のノーツの連続が指定した長さからランダムで選ばれるようになります
入力の仕方　例)2,3,6,2,4　このように半角数字をカンマで区切ってください、この場合だと2連続の配色が40%、3と4と6連続が20%ずつ出ます

"作る数"　同時に複数個作れます、100以上を入れても問題ないです、空欄の場合は1になります

"後ろにつく数字の始点を変えられます"　()の中の数字がここから始まります　

"生成する譜面に元の難易度名を残さない"　難易度名が長くなりすぎると曲選択画面から見切れてしまう可能性があります

"同名のファイルが既にある場合上書きする"　上書きの場合は曲選択画面に何も出ませんがプレイしたら新しい譜面になっています、チェックがない場合は既存の譜面がなくなるまで後ろの番号に+1します



～その他～

"履歴から選択する"　一度選択した譜面はここから選べるようになります

左の枠は上が変更前のファイルの中身、下が変更後のファイルの中身です、表示する意味は特にないです

LRandomのLはLimitedが長くて不便なので短くしたものです
